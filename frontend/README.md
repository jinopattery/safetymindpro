# SafetyMindPro Frontend

React-based frontend for SafetyMindPro safety analysis platform.

## Features

- **Dashboard**: Overview of all FMEA and FTA analyses
- **FMEA Management**: Create, view, and manage FMEA analyses
- **Fault Tree Analysis**: Create and manage fault trees
- **Risk Matrix**: Visual risk assessment matrix
- **Export**: Export FMEA data to Excel/CSV

## Installation

```bash
cd frontend
npm install
```

## Running the App

Make sure the backend is running first, then:

```bash
npm start
```

The app will open at http://localhost:3000

## Building for Production

```bash
npm run build
```
